# Pull Request Skill

## When to use this skill
- ユーザが、現在のブランチのプルリクを依頼した場合

## How to use it
- ユーザのプルリク依頼を受け、`gh` コマンドを使って、現在のブランチを `main` ブランチ向けにプルリクを作成する
- プルリクを作成する際、分岐前のブランチからの作業差分のコメントを確認し、適切にプルリクの説明を付与する
- プルリクを作成する際は、ユーザがコミットを済ませている前提のため、新規のコミットの制御は不要である
